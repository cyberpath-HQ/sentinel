## CLI Installation

### Linux (glibc)

**Debian/Ubuntu:**

```bash
# Download
wget https://github.com/cyberpath-HQ/sentinel/releases/latest/download/sentinel_1.0.0-beta.1_amd64.deb

# Install
sudo dpkg -i sentinel_1.0.0-beta.1_amd64.deb
```

**RHEL/CentOS/Fedora:**

```bash
# Download
curl -L https://github.com/cyberpath-HQ/sentinel/releases/latest/download/sentinel-1.0.0-beta.1.x86_64.rpm -o sentinel.rpm

# Install
sudo rpm -i sentinel.rpm
```

**Arch Linux:**

```bash
# Install from AUR
yay -S sentinel-bin
# Or manually
git clone https://aur.archlinux.org/sentinel-bin.git
cd sentinel-bin
makepkg -si
```

### Linux (musl / Alpine)

**Alpine:**

```bash
# Add repository
echo "@cyberpath https://github.com/cyberpath-HQ/sentinel/releases/latest/download/sentinel-1.0.0-beta.1.apk" | sudo tee -a /etc/apk/repositories

# Install
sudo apk add --repository @cyberpath sentinel
```

### macOS

**Manual:**

```bash
# Download
curl -L https://github.com/cyberpath-HQ/sentinel/releases/latest/download/sentinel-macos-universal -o sentinel

# Make executable and install
chmod +x sentinel
sudo mv sentinel /usr/local/bin/sentinel
```

### Windows

1. Download the appropriate .exe file
2. Rename to `sentinel.exe`
3. Add to your PATH or move to a directory in PATH

## Language Bindings

### Python Bindings

**From PyPI:**

```bash
pip install sentinel-dbms
```

**Source:**

```bash
git clone https://github.com/cyberpath-HQ/sentinel.git
cd sentinel/bindings/python
pip install maturin
maturin develop
```

### Node.js Bindings

**From npm:**

```bash
npm install @cyberpath/sentinel
```

**Source:**

```bash
git clone https://github.com/cyberpath-HQ/sentinel.git
cd sentinel/bindings/nodejs
npm install
npm run build
```

### C/C++ Bindings

**Pre-built packages:**

Linux (glibc):

```bash
wget https://github.com/cyberpath-HQ/sentinel/releases/latest/download/sentinel-cxx-linux-x86_64.tar.gz
tar -xzf sentinel-cxx-linux-x86_64.tar.gz
sudo cp -r include/sentinel-cxx /usr/local/include/
sudo cp lib/libsentinel-cxx.so /usr/local/lib/
sudo ldconfig
```

macOS:

```bash
curl -L https://github.com/cyberpath-HQ/sentinel/releases/latest/download/sentinel-cxx-macos-universal.tar.gz | tar -xzf
sudo cp -r include/sentinel-cxx /usr/local/include/
sudo cp lib/libsentinel-cxx.dylib /usr/local/lib/
```

Windows: Download from releases page and extract to `C:\Program Files\sentinel-cxx`

**Source:**

```bash
git clone https://github.com/cyberpath-HQ/sentinel.git
cd sentinel/bindings/cxx
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --target sentinel-cxx
cmake --install build --prefix /usr/local
```

## Docker Images

Official Docker images are available:

| Tag             | Description                |
| --------------- | -------------------------- |
| `latest`        | Latest stable release      |
| `x.y.z`  | Specific version           |

```bash
# Pull the image
docker pull cyberpath/sentinel:latest

# Run the CLI
docker run --rm -v $(pwd):/data cyberpath/sentinel:latest --help

# Use as a base image
FROM cyberpath/sentinel:latest
```

## Requirements

### CLI

| Platform     | Minimum Version                        |
| ------------ | -------------------------------------- |
| Linux        | glibc 2.17+ (CentOS 7+, Ubuntu 14.04+) |
| Linux (musl) | Alpine 3.8+                            |
| macOS        | 10.14+ (Mojave)                        |
| Windows      | 10+                                    |

### Python Bindings

- Python 3.8 or higher

### Node.js Bindings

- Node.js 18.0.0 or higher

### C/C++ Bindings

- C++11 compiler (suggested GCC 8+, Clang 7+ or MSVC 2019+)
- CMake 3.16+
